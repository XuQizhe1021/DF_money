# 用户手册（M4交付版）

本系统用于《三角洲行动》子弹行情的本地分析与辅助决策，不登录游戏账号、不自动交易。

## 1. 快速开始

1. 安装 Python 3.11+ 与 Node.js 20+
2. 在项目根目录执行：

```bash
python -m venv .venv
. .venv/Scripts/activate
pip install -r requirements.txt
copy .env.example .env
cd frontend
npm install
cd ..
```

3. 启动后端：

```bash
python -m backend.app
```

4. 启动前端：

```bash
cd frontend
npm run dev
```

5. 浏览器访问 `http://127.0.0.1:5173`

## 2. 功能说明

### 2.1 行情看板

- 刷新行情：拉取最新价格与更新时间
- 7日涨跌榜：显示涨幅Top3和跌幅Top3
- 检查提醒：手动触发持仓提醒评估并展示通知

### 2.2 走势分析

- 可选择子弹与时间范围（7日/30日）
- 支持图表缩放与平滑查看长序列

### 2.3 AI建议

- 选择子弹后触发分析
- 若AI不可用，系统自动回退规则分析并返回建议

### 2.4 持仓管理

- 新增持仓支持“按现价填充”
- 持仓列表显示现价、盈亏与状态
- 支持编辑、卖出、删除

## 3. 常见问题

### Q1：为什么没有行情数据？

- 先点击“刷新行情”
- 检查 `.env` 中 API 地址配置
- 若远端波动，可开启 `MOCK_ON_FAILURE=true` 保证本地可用

### Q2：AI建议为什么是规则分析？

- 可能未配置AI Key，或触发了频率限制，或AI服务暂时不可用
- 系统会自动回退规则分析，属于正常保护行为

### Q3：提醒为什么没触发？

- 确认持仓阈值是否合理
- 点击“检查提醒”手动触发一次
- 检查提醒冷却时间配置是否过长

## 4. 使用限制

- 数据存在延迟，仅用于辅助判断
- 不提供自动下单能力
- 默认本地存储，不做云端托管
