# Release Notes - M4 集成测试与优化交付版

## 版本定位

- 版本：M4
- 状态：可交付候选版本
- 目标：稳定性、性能、可用性、文档与部署能力完善

## 主要变更

1. 新增全链路集成测试，覆盖正常流、异常流、边界流。
2. 新增后端聚合接口 `/api/ammo/change-ranking`，替代前端多次历史查询。
3. 持仓列表与提醒列表支持分页参数（limit/offset），并优化查询路径。
4. 后端热点最新价查询增加短TTL缓存，降低重复SQL压力。
5. 图表交互优化：历史数据短缓存、防抖刷新、长序列渲染参数优化。
6. 持仓新增体验优化：支持“按现价填充”与现价参考展示。
7. 完成用户手册、API接入教程、测试报告、性能报告与部署脚本。

## 兼容性与注意事项

- 外部行情源不稳定时，建议启用 `MOCK_ON_FAILURE=true`。
- AI能力依赖用户配置的兼容服务与密钥，失败会自动回退规则分析。
- 本版本仍为本地部署形态，不包含自动交易能力。

## 回退策略

- 新增优化均为小步变更，可按文件级快速回退。
- 若聚合接口不满足预期，可回退到旧版前端多请求方案（不推荐）。
- 若缓存策略影响调试，可临时关闭前端缓存（forceRefresh）进行定位。
