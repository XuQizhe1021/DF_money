# 分阶段开发 Prompt 使用说明

本目录提供 4 个“可直接复制即用”的阶段化 Prompt，严格对应[计划书](file:///e:/money/计划书.md)中的四个开发阶段。你只需按阶段顺序，将对应文件中的 `PROMPT START` 到 `PROMPT END` 全部内容复制给 AI 执行即可。

## 使用顺序

1. 阶段1：基础搭建（后端骨架、数据采集、数据库）
2. 阶段2：核心功能开发（AI分析、持仓、监控提醒、联调测试）
3. 阶段3：前端开发（Vue3页面、图表、通知、交互）
4. 阶段4：集成测试与优化（全链路验证、性能与交付）

## 执行规则

- 同一阶段内，要求 AI 先给实施方案，再落地代码与测试，最后给验证结果
- 每阶段完成后，先确认验收标准全部通过，再进入下一阶段
- 若出现外部 API 不稳定或鉴权问题，按阶段 Prompt 中“风险与回退策略”执行
