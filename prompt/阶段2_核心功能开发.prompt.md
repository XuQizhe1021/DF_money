【项目固定上下文】

- 前置条件：阶段1已完成基础后端、数据库、数据采集与基础 API

【本阶段目标（M2）】

【阶段范围（只做这些）】
1) AIAnalyzer 模块
   - 支持用户配置 API Key 与模型名

3) 价格监控与提醒模块
   - 规则：当前价格 > 成本价 * (1 + 阈值) 触发提醒
   - 支持页面通知数据接口与控制台提醒
   - 支持每个持仓单独阈值或全局默认阈值
4) API 联调完善
   - 增加 /api/analysis/*、/api/holdings/*、/api/alerts/*
   - 统一错误码、错误消息、输入校验
5) 测试与稳定性
   - 单元测试 + 接口测试，关键路径覆盖率 >70%
   - 异常场景：AI超时、空数据、数据库冲突、非法参数
